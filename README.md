## Features
- **User Management**  
  Create, update, and delete user profiles
- **Skill Tracking**  
  Log skills with descriptions and goals
- **Learning Sessions**  
  Record lessons with duration and notes
- **Database Integration**  
  Uses SQLite + SQLAlchemy for data storage
